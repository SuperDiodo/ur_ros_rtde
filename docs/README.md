# 📖 Documentation

This folder contains the documentation for `ur_ros_rtde`, organized into multiple Markdown files.  
Each document is briefly described below.

---

## Contents

- **[Plugin System](plugin_system.md)**  
  Overview of the plugin architecture and description of the available plugins.

- **[Available Plugins](plugin_list.md)**

- **[Extending the Control Script](extending_the_control_script.md)**
